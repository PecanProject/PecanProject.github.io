This release has a number of bug fixes and added functionality. All can be seen [listed here](https://github.com/PecanProject/pecan/releases/tag/v1.6.0).

<!--truncate-->

## Major additions include:

- A data ingest SHINY app that allows users to upload and register data from their own local machine or from the DataOne
- Preliminary containerized modules of PEcAn using Docker
- Preliminary automated, Daily Forecasting using NOAA GEFS and the Sipnet model

The VM can be downloaded at https://opensource.ncsa.illinois.edu/projects/artifacts.php?key=PECAN (note: VM has been deprecated, so this link may no longer work).
